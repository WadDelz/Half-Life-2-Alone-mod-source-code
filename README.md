# The Official - Half-Life 2: Alone mod source code
This is the official github page for half-life 2 alone mod (https://www.moddb.com/mods/half-life-2-alone-mod). It contains all the code for the features of the mod + some unreleased/undocumented stuff.

Alot of the code for the mod is pretty crappy (there used to be alot of hardcoding that i've tried to remove) BUT thats because alot of the code was made when i was just learning programming. I have however tried to fix up some of the files to make them more readable.

This uses the source sdk 2013 singleplayer code (https://github.com/ValveSoftware/source-sdk-2013) as the base for the mod.

If you have any questions or concerns. Please let me know at ytwaddelz@gmail.com OR by dm'ing me on moddb.
